# Microsoft MakeCode helper extension for the micro:bit machine learning extension

This extension is leveraged by [pxt-microbit-ml](https://github.com/microbit-foundation/pxt-microbit-ml) to create meaningful examples in the help documentation that would otherwise be impossible.
